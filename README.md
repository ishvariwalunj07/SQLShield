SQLShield – SQL Injection Detection & Prevention System

SQLShield is a cybersecurity-focused web application developed using Flask and MySQL that demonstrates how SQL Injection attacks occur and how they can be prevented using secure coding practices.

The project includes both vulnerable and secure authentication systems to compare insecure query handling with parameterized queries. It also provides an interactive SQL Injection Playground for testing payloads, an Admin Panel for monitoring attack logs, and a Reporting Module for analyzing attack activity.

🚀 Features
🔐 Vulnerable Login System (SQL Injection Demonstration)
✅ Secure Login using Parameterized Queries
🧪 SQL Injection Playground for Payload Testing
🛡️ Admin Panel for Attack Monitoring
